# 启动方式

进入ddn-font-end文件夹中

```bash
yarn install 

yarn start 

```

等待启动完成后使用Expo app扫码或者adb进行程序的启动。


# 打包方式

在ddn-font-end文件夹中：

```bash
yarn eas build //默认打包双端

yarn eas build --profile preview2 --platform android //默认只打包Android，生成apk格式安装包


```
